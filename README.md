# Название
"Проектор"

# Описание

«Проектор» — мобильное приложение для учёта проектов, разработанное в рамках учебной практики. Позволяет создавать, просматривать, редактировать и удалять проекты с указанием ключевых параметров: названия, описания, дат начала и окончания, прогресса.

# Установка

Чтобы установить приложение:

Скачайте файл app-debug и подтвердите установку. Найдите иконку приложения на главном экране или в меню приложений и запустите «Projector».

# Использование

Данное приложение позволяет вести учёт проектов: создавать, просматривать, редактировать и удалять их. При создании проекта пользователь открывает всплывающее окно нажатием кнопки «+» на главном экране и заполняет следующие поля: название проекта, описание, дата начала, дата окончания и прогресс (в диапазоне от 0 до 100 %). После ввода данных проект сохраняется нажатием кнопки «Сохранить» либо отменяется кнопкой «Отмена». Все созданные проекты отображаются списком на главном экране приложения — там видны их названия и текущий процент прогресса. Для редактирования достаточно нажать на строку нужного проекта в списке, внести необходимые изменения в открывшемся окне и подтвердить их кнопкой «Сохранить». Чтобы удалить проект, нужно удерживать нажатие на его строке в списке более 2 секунд, после чего подтвердить удаление — запись мгновенно исчезнет из общего перечня. Приложение разработано в рамках учебной практики и ориентировано на простой и быстрый учёт проектных задач без избыточного функционала.
